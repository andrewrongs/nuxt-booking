## 目錄架構
- src
    - public # 不需編譯靜態檔
    - assets # 靜態檔案
    - components # 元件
    - layouts # 版面配置
    - composables # 自訂組合式函式
    - locales # i18n
    - plugins # 三方套件
    - middleware # 中介
    - store # pinia
    - pages # 頁面
    - types # 型別
    - utils # 工具
    - api # API
    
## 指令

### pnpm
- 安裝步驟
```
pnpm i 
```

- 啟動本地端
```
pnpm run dev 
```

- 打包專案
```
pnpm build 
```
